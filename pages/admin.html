📖 TEMPLATE GUIDE — HOW TO CONDUCT
🏃 3 ROUNDS — STANDARD (15-30 min)
Best for: Quick classroom quizzes, post-lecture assessments
Structure: R1: Qualifiers (Preliminary) → R2: Knowledge (Selection) → R3: Finals (Final)
Time/Question: 15 min = 60s | 30 min = 120s
How to run: Start quiz → Begin R1 → Let all teams answer → End R1 → Eliminate bottom teams → Begin R2 → End R2 → Begin Finals
📝 5 ROUNDS — COMPREHENSIVE (30-45 min)
Best for: Department-level competitions, inter-class quizzes
Structure: R1-R2 (Preliminary) → R3-R4 (Selection) → R5 (Final)
Time/Question: 30 min = 72s | 45 min = 108s
How to run: Start → Run Preliminary rounds → Eliminate weak teams after R2 → Run Selection → Eliminate again after R4 → Grand Finals in R5
🏆 10 ROUNDS — MARATHON (45-60 min)
Best for: College-level tournaments, inter-department events
Structure: R1-R3 (Preliminary) → R4-R6 (Selection) → R7-R10 (Final)
Time/Question: 45 min = 54s | 60 min = 72s
How to run: Start → Run 3 Preliminary rounds → Eliminate bottom 50% → Run 3 Selection rounds → Keep top teams only → 4 Final rounds for championship
🎯 15 ROUNDS — TOURNAMENT (60 min)
Best for: Full-day tournaments, inter-college competitions
Structure: R1-R5 (Preliminary) → R6-R10 (Selection) → R11-R15 (Final)
Time/Question: 60 min = 48s per question
How to run: Same as marathon but with 3 distinct elimination gates
📋 STEP-BY-STEP ADMIN WORKFLOW
  1. Select Template — Choose rounds & time above, click APPLY
  2. Add Questions — Go to Questions tab, add 5 per round (or import CSV)
  3. Create/Auto-Generate Teams — Use Teams tab → Auto Setup or manual
  4. Start Quiz — Go to Quiz Control → Click START QUIZ
  5. Begin Each Round Manually — Click BEGIN ROUND (never auto-starts)
  6. Monitor Answers — Watch live answers, timer, pass chains
  7. End Round — Click END ROUND when done
  8. Eliminate Teams — At stage transitions, use 🚫 ELIMINATE TEAMS
  9. Next Round — Click NEXT ROUND (stage gate confirms transitions)
  10. Repeat — Until all rounds complete → Quiz Finished!
💡 KEY RULE: Rounds NEVER start automatically. You always have full manual control. Use the stage system (Preliminary → Selection → Final) to organize your elimination process.
