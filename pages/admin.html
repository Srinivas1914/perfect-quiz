📖 ADMIN MASTER GUIDANCE
📖 TEMPLATE GUIDE — HOW TO CONDUCT
🏃 3 ROUNDS — STANDARD (15-30 min)
Best for: Quick classroom quizzes, post-lecture assessments
Structure: R1: Qualifiers → R2: Knowledge → R3: Finals
How to run: Start quiz → Begin R1 → End R1 → Eliminate bottom teams → Begin R2 → End R2 → Begin Finals
📝 5 ROUNDS — COMPREHENSIVE (30-45 min)
Best for: Department-level competitions, inter-class quizzes
Structure: R1-R2 (Preliminary) → R3-R4 (Selection) → R5 (Final)
How to run: Run Preliminary → Eliminate weak teams after R2 → Run Selection → Grand Finals in R5
📋 STEP-BY-STEP ADMIN WORKFLOW
  1. Select Template — Choose rounds & time, click APPLY
  2. Add Questions — Go to Questions tab, add or import CSV
  3. AI Generate — Use "🤖 AI GENERATE" for quick set creation
  4. Start Quiz — Go to Quiz Control → Click START QUIZ
  5. Conduct Rounds — Click BEGIN ROUND for each stage
  6. Eliminate Teams — Use the 🚫 ELIMINATE button at stage transitions
  7. Finish — Complete all rounds to generate final scores
💡 PRO TIP: The Live Feed button in the header opens a dedicated window for your projector. Extend your display and move the Live Feed window there!
